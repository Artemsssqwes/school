# 🛠 План розробки проєкту: crash assistant
 
## 1. Концепція продукту
- **Тип додатку:** Десктоп
- **Основна ідея:** understandment of why one of your programs crashed with logs from said program
 
## 2. Основний функціонал (Features)
- [ ] get text (crash log) from user
- [ ] read said crash log and identify the reason why the crash has happed
- [ ] Виводити результат
 
## 3. Структура даних
| Назва змінної | Тип даних | Для чого потрібна |
| :--- | :--- | :--- |
| `user_input` | `str` | Ввід тексту |
| `is_active` | `bool` | Контроль циклу |
